## Key Features

* Memtables, including an active memtable and an immutable memtable
  - implemented both in skip-list and red-black-tree
* Sorted-string-table
  - implemented both in B+ tree 
  - and sparse index + bloom filter
* Linux/C++ 
* supported by CMake
* one-layer sstable so far
